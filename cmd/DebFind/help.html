How to choose Sections or Tags:
  • To choose one simply Click it.
  • To choose two or more that are consecutive, Click the first then Shift+Click the last; this will select the two clicked and any in between.
  • To choose two or more nonconsecutive, Ctrl+Click each one.
  • To unchoose (i.e., unselect) Ctrl+Click it.
Alternatively, press Alt+S to navigate to the Sections list, then use the Up and Down arrows to move about and press Spacebar to choose or unchoose. The same can be done for Tags by pressing Alt+T to navigate to the Tags list, then using the Up and Down arrows and Spacebar.

To find packages:

  • For Sections either choose one or more Sections in which case packages from any chosen section will be found, or don't choose any sections at all (Clear All); in which case packages from any section at all will be chosen.
  • For Tags chose either no Tags (Clear All), or one or more Tags. Then click All if each package must have all the given tags or Any if each package may have any of the given tags.
  • For Words optionally enter one or more words: these are searched for case-insensitively in each package's name and description. Then click All if each package must have all the given words or Any if each package may have any of the given words.
  • Click Find; this will find any packages in any of the chosen Sections (or any Section if none are chosen), and which has all or any of the given Tags (depending on whether All or Any is checked and if any Tags have been chosen at all), and which has all or any of the given Words (depending on whether All or Any is checked and if any Words have been entered).
